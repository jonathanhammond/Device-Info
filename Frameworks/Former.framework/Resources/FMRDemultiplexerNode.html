Demultiplexer

The Demultiplexer patch switches a single input between multiple outputs based on an index value.

  • Index
    The destination index.
  • Input
    The value to be passed to the active destination.
  • Default
    The value to pass to all other destinations.
  • Destination x
    The input value or the default value depending on the destination index.
  • Destinations
    The number of destination outputs.
