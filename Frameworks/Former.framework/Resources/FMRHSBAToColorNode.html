HSBA to Color

The HSBA to Color patch converts hue, saturation, brightness, and alpha components to a color. The outputs are normalized values between 0 - 1.

  • Hue
    The color hue.
  • Saturation
    The color saturation.
  • Brightness
    The color brightness.
  • Alpha
    The opacity component.
  • Color
    The resulting color.
