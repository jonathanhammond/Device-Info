Simple Image Fill

The Simple Image Fill renders an image view that completely fills the entire parent view without borders or whitespace. It scales the image while respecting its aspect ratio, and maskes the image to size of the parent view. A common use case would be a fullscreen background image. It is a group patch and can be entered and customized.

  • Image
    The input image.
  • Opacity
    The opacity of the view. Range from 0 to 1.
  • Filter
    A filter to apply to the image.
