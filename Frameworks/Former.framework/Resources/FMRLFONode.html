LFO

The LFO patch simulates a low frequency oscillator. It is commonly used for animations and other effects.

  • Sine
  • Cosine
  • Square
  • Wave
    The wave type to use for the oscillator.
  • Period
    The amount of time in seconds for the LFO to complete one cycle.
  • Phase
    A normalized value ranging between 0 - 1 that represents a fraction of the period.
  • Amplitude
    The amplitude of the wave.
  • Offset
    A number used to displace the wave output value.
  • Value
    The current output value of the LFO.
