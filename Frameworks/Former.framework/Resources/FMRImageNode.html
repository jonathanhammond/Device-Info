Image

The Image patch holds image data and can be connected to an Image View to display it. You can drag images onto the document to automatically create an Image patch. The image data can also be set in this patch's settings.

  • Width
    The width of this image.
  • Height
    The height of this image.
  • Image
    The image data, which is typically connected to an Image View.
  • Image
    The input image.
