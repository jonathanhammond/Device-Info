Image Controls

The Image Controls patch adjusts the brightness, contrast, saturation, highlights, and shadows of an image.

  • Filter
    A filter to combine with the image controls filter.
  • Brightness
    Adjusts the brightness of an image. Range from 0 to 1. Midpoint is 0.5.
  • Contrast
    Adjusts the contrast of an image. Range from 0 to 1. Midpoint is 0.25.
  • Saturation
    Adjusts the saturation of an image. Range from 0 to 1. Midpoint is 0.5.
  • Highlights
    Saves the highlights of an image. Range from 0 to 1. Starting point is 1.
  • Shadows
    Saves the shadows of an image. Range from 0 to 1. Starting point is 0.
  • Filter
    The image controls filter.
