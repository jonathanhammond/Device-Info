Linear Out, Slow In

Linear Out, Slow In is one of the animation curves used in Material Design. Similar to Fast Out, Slow In, except the initial part of the curve is linear. This curve should be used for motion when entering frame or when fading in from 0.0 opacity.

  • Progress
    A number from 0-1 that represents the progress of the animation.
  • Progress
    A number from 0-1 that represents the progress of the animation with the curve applied to it.
