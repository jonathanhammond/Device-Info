Reverse Transition

The Reverse Transition patch takes a value (without clamping it) and scales such that the specified input minimum is mapped to an output of 0.0 and the input maximum is mapped to an output of 1.0.

  • Input
    The input value.
  • Input Min
    The minimum value for the input range.
  • Input Max
    The maximum value for the input range.
  • Output
    The output value.
