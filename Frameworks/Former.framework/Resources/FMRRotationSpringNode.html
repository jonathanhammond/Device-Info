Rotation Spring

The Rotation Spring patch simulates an n-dimensional rotation spring. It will always attempt to find the shortest distance from the current position to the rest position, including if this means changing directions.

  • Strength
    The strength of the spring.
  • Damping
    The damping or friction on the spring.
  • Sampling
    A boolean value that indicates whether the patch is sampling.
  • Rest
    The resting position of the spring.
  • Position
    The current position of the spring and view.
  • Dimensions
    The number of dimensions on the spring.
