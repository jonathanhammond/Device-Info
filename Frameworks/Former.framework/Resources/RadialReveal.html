Radial Reveal

The Radial Reveal implements the fill transition used for Material Design visual continuity.

  • X Position
    The X position at which the transition begins.
  • Y Position
    The Y position at which the transition begins.
  • Color
    The background color of the transition.
  • Image
    An optional image to display within the transition.
  • Image Width
    The width of the image.
  • Image Height
    The height of the image.
  • Fade Delay
    A number from 0-1 that determines when to begin fading the image. At 0.0, the fade begins immediately when the animation begins. 0.5 will start it halfway.
  • Progress
    A number from 0-1 that represents the progress of the animation.
