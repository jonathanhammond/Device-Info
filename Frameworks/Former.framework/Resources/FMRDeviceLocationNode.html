Device Location

The Device Location patch outputs the device's GPS coordinates.

  • Locate
    Trigger this signal input to start outputting location data.
  • Accuracy
    Location data is collected within a given distance range. This input controls how accurate you wish the location data to be. Any number (in meters) can be used.
  • Latitude
    The latitude in degrees. The northern hemisphere uses positive values, while the southern hemisphere has negative ones.
  • Longitude
    The longitude in degrees. East of the prime meridian has positive values, and west of the prime merdian has negative ones.
  • Locked
    This value is YES if the patch is collecting data.
