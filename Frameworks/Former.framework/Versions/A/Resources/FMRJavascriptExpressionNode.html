JavaScript Expression

The JavaScript Expression patch processes inputs with a JavaScript function.

  • Expression
    The JavaScript function. Any input given to this function will appear as inputs on the patch. The function must be called 'expression'.
  • Variables
    The variables as specified in the JavaScript function.
  • Result
    The return value of the JavaScript function.
  • Error
    If there was an error parsing or processing the Javascript function, it will be outputted here.
