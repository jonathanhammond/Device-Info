Fast Out, Slow In

Fast Out, Slow In is one of the animation curves used in Material Design. Similar to Fast Out, Slow In, except the ending part of the curve is linear. This curve should be used for motion when exiting frame or when fading out to 0.0 opacity.

  • Progress
    A number from 0-1 that represents the progress of the animation.
  • Progress
    A number from 0-1 that represents the progress of the animation with the curve applied to it.
