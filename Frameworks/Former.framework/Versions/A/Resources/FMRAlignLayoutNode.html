Align Layout

The Align Layout patch makes it easy to position views relative to another view. You can use this patch to center views within their superview, or align an align a view along the edge of another view. In order for alignment to work properly, you must connect both the position and anchor point outputs to the view you wish to align.

  • Size
    The width, height, and depth of the view you are aligning with.
  • X Align
    Align the view to the left, center, or right with respect to the given size. You can also enter any value from 0-1.
  • Y Align
    Align the view to the top, center, or bottom with respect to the given size. You can also enter any value from 0-1.
  • Z Align
    Align the view to the front, center, or back with respect to the given size. You can also enter any value from 0-1.
  • Position
    The position of the aligned view.
  • Anchor
    The anchor point of the aligned view.
