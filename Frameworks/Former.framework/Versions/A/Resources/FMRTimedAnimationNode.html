Timed Animation

The Timed Animation patch is a controller for animations. Use it to control the duration and motion curve of the animation.

  • On/Off
    If true, animate towards a progress of 1.0. If false, animate towards a progress of 0.0
  • Duration
    The time (in seoncds) begin the start and end of the animation.
  • Curve
    The type of motion curve to apply to this animation. The standard material curves are included, as well as linear.
  • Progress
    A number from 0-1 that represents the progress of the animation.
