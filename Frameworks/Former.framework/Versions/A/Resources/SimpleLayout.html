Simple Layout

The Simple Layout makes it easy to position views relative to the parent view. The X, Y offsets allow simple positioning in respect to the alignment. It is a group patch and can be entered and customized.

  • Offset
    The offset of the resulting alignment.
  • Align
    Align the view to the left, center, or right relative to the parent view. The default is center.
  • Position
    The position of the aligned view.
  • Anchor Point
    The anchor point of the aligned view.
