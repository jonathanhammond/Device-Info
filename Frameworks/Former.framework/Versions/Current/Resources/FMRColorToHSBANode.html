Color to HSBA

The Color to HSBA patch converts a color to its hue, saturation, brightness, and alpha components. The outputs are normalized values between 0 - 1.

  • Color
    The input color.
  • Hue
    The color hue.
  • Saturation
    The color saturation.
  • Brightness
    The color brightness.
  • Alpha
    The opacity component.
