Material Icon

The Material Icon patch outputs images of the Material Design system icon set. Icons are indexed by name. A full list can be found in the icon font repository.

  • Name
    The name of the icon.
  • Color
    The color of the icon..
  • Width
    The width of this image. System icons are always 24pt in width.
  • Height
    The height of this image. System icons are always 24pt in height.
  • Image
    The image data, which can be connected to an App Bar, Floating Action Button, Image View, or any other patch that has an image or icon input.
