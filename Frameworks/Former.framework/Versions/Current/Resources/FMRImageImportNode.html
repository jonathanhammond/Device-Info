Image Importer

The Image Importer fetches image data from a URL and outputs an image.

  • URL
    The patch will fetch the image data from this URL.
  • Load
    When triggered, this input will start the JSON request.
  • Size
    The dimensions of the image that has been downloaded.
  • Image
    The image that has been downloaded.
  • Error
    If an error occurred during the request, it will be outputted here.
  • Done
    When the request is complete, this value will be set to YES.
  • Scale
    Use this as scale for all imported images.
