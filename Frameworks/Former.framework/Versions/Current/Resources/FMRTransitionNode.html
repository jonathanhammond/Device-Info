Transition

The Transition patch takes a value (without clamping it) and scales such that an input of 0.0 is mapped to the output minimum and an input of 1.0 is mapped to the output maximum.

  • Input
    The input value.
  • Value Min
    The minimum value for the output value range.
  • Value Max
    The maximum value for the output value range.
  • Value
    The output value.
