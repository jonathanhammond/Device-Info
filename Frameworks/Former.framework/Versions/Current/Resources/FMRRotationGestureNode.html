Rotation Gesture

The Rotation Gesture patch recognizes a two finger rotation gesture on a view. It is commonly used with the Z-rotation port on a view.

  • Reset
    Resets the gesture.
  • Rotation
    The rotation offset.
  • Gesture
    The gesture connection port used to link the gesture to a view.
  • Rotation
    The current rotation of the rotation gesture view.
  • Active
    A boolean value that represents whether the patch has recognized a rotation gesture.
