Simple Spring

The Simple Spring simulates a spring animation. It is gracefully interruptible and is commonly used to create a bouncy animation. It is a group patch and can be entered and customized.

  • Value
    The spring position input. I.e. where you want to spring to end up.
  • Speed
    The speed defines how quickly the animation appears to happen. Range from 0 (slowest) to 1 (fastest).
  • Stiffness
    The stiffness defines how stiff the animation appears to behave. Range from 0 (loosest) to 1 (stiffest).
  • Value
    The resulting position of the spring. I.e. where the spring ends up.
