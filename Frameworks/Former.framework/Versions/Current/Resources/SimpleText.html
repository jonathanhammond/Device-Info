Simple Text

The Simple Text helps draw text to the screen and assists with laying out the element via aligning. The patch defaults to the center of the parent view. It is a group patch and can be entered and customized.

  • Position
    The position of the view.
  • Align
    The alignment of the view in relation to the parent view. The default is the center.
  • Text
    The text to draw.
  • Font
    The name of the font to use. Supports all fonts bundled with iOS, also supports Roboto.
  • Size
    The font size in points.
  • Color
    The font color.
  • Opacity
    A normalized value between 0 - 1 that represents the opacity.
  • Wrap Text
    Wrap the text to a width and height.
  • Wrap Width
    The width of the text wrapping.
  • Wrap Height
    The height of the text wrapping.
