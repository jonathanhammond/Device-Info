Floating Action Button

The Floating Action Button patch represents the Material Design floating action button. It can be displayed in a regular size or a mini size. It outputs the down state, but cannot be disabled.

  • X Position
    The X position of the component.
  • Y Position
    The Y position of the component.
  • Mini
    If true, use the mini size for this component.
  • X Anchor
    The X anchor point of the component.
  • Y Anchor
    The Y anchor point of the component.
  • Background Color
    The primary color of this component. The weight is automatically chosen based on the state of the button.
  • Icon
    The icon to display inside the button.
  • Down
    A boolean value that represents whether the patch has recognized a touch.
